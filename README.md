## Permute 到底能做哪些事？很多人只用了 10%

很多人用 Permute 只是转转视频格式，但这个工具实际能做的事情远不止于此。

**格式覆盖范围**：视频支持 MP4、MKV、MOV、AVI、HEVC、ProRes、WebM、AV1 等几十种格式；音频支持 MP3、AAC、FLAC、WAV、Apple Lossless 等；图片支持 HEIC、AVIF、WebP、PNG、JPEG、TIFF、BMP、GIF……基本上你在工作中遇到的奇怪格式，它都认识。

**基本转换操作**很简单：把文件拖进 Permute 窗口，右侧出现格式预设列表，选一个，点开始转换。批量文件也行，一次拖几十个进去，排队处理。

**编辑功能**比很多人想象的更丰富。裁剪、旋转、翻转、调速、加字幕、替换音轨、调整音量——这些操作都可以在 Permute 里直接完成，不用额外开别的软件。

**创意工坊（Workshop）**是 Permute 的彩蛋区。这里藏了：文本转语音（把文字生成音频旁白）、图片 OCR 识别提取文字、iPhone 铃声制作、视频帧导出为图片序列、元数据编辑、PDF 页面导出为图片。很多用户用了很久才发现这块功能。

**合并与拆分**：多段视频可以直接拼接成一个文件，也可以按时间点拆开。拆分的时候支持无重编码导出独立轨道，速度快而且不损失画质。

转换速度方面，Permute 利用 Apple Silicon 的硬件编码能力加速处理，ProRes 和 HEVC 的转换速度明显快于纯软件方案。从实际使用体验来看，M 系列芯片上批量转换一批家庭录像，效率相当高。

## 如何激活 Permute 4？两步搞定

购买后收到的是一串激活码，激活过程如下：

1. 打开 Permute 4，在菜单栏点击「Permute 4 > 输入许可证代码」，弹窗中点击「输入许可证代码」按钮
2. 在输入框填写你的邮箱账号，粘贴激活码，点击「激活许可证」完成

一个授权码支持最多 3 台设备同时激活。更换电脑时，先在旧设备里反激活（通过 Permute 菜单操作），再在新设备激活即可。

## Permute vs 其他 Mac 格式转换方案：哪个适合你？

不是所有人都需要 Permute。这里做个直接的对比，帮你判断。

**Permute vs HandBrake（免费）**：HandBrake 完全免费，压缩参数控制更精细，是发烧友的首选。但 HandBrake 只做视频，不支持音频和图片；界面参数繁多，新手上手需要时间；而且处理完整个流程比较费力。Permute 的优势是全格式覆盖、上手时间几乎为零，批量操作更快。

**Permute vs 在线转换工具**：免费在线工具的常见问题是文件大小限制（通常 200MB 以内）、上传隐私风险、网速依赖、转换后画质不稳定。Permute 在本地运行，无文件大小限制，不联网，适合需要处理大文件或私密素材的场景。

**简单总结**：偶尔转一两个小视频，在线工具够用；经常需要批量处理、或者涉及音频和图片格式、或者文件较大的场景，Permute 更合适。它 $14.99 买断，不是按月付费，长期成本非常低。

不通用。Permute 4 是全新版本，需要新的授权码。购买 Permute 3 的用户在升级 macOS 26 时可以获得免费或折扣价升级，具体见开发商升级页面。
